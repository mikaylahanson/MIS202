Learning Objectives

These instructions will show you how to start and run Node-RED on a Raspberry Pi.
This software provides a no-code / low-code environment for creating applications (flows) on a Raspberry Pi.

More information on Node-RED is available online.

After completing this assignment you will be able to:

  1. Start Node-RED on a Raspberry Pi
  2. Connect to Node-RED using a browser
  3. Modify a Node-RED flow

Exercise 1-Verify Node-RED Installation

Starting Node-RED

  1. These steps are performed on the Raspberry Pi
  2. In the Raspberry Pi desktop menu bar, click the Node-RED icon to start the service if it is not already running:

    start Node-RED

  3. After a few minutes, the Terminal window should display a message that the Node-RED service is running, on port :1880
    Note the RPi IP address as you will need it in the next exercise.
  4. If Node-RED does not start up, begin trouble-shooting; update the RPi, then reboot and try again.
  5. Open the browser and go to: 127.0.0.1:1880 (if it is not already running)
  6. If the webpage does not appear, troubleshoot and resolve any issues.

Exercise 3-Node-RED Flows
  1. These steps are performed on your computer, or the Raspberry Pi.
  2. Open a browser and go to: 10.19.x.x:1880 (enter the RPi IP address from Exercise 1)
  3. Retrieve the Node-RED flow, BuzzerDigital.json from Professor Paulson's OneDrive, ReadOnly, IoT, NodeRED_Flows folder, and place it in your 'Downloads' folder:

    Node-RED Digital Buzzer

  4. In Node-RED from the menu in the upper-right corner choose Import>select a file to import.
    Navigate to your Downloads folder and select 'BuzzerDigital.json'
    Make sure to select 'Import to new flow':

    Import Buzzer Flow

  5. The flow code looks like the following during import:

    Buzzer Flow Code

  6. Double click the Buzzer node to view the Properties page:

    Buzzer Node Properties

  7. Examine the properties.
    Note that the 'Digital Pin' setting connects the Node-RED flow to the physical digital buzzer which is connected to digital pin 3 on the GrovePi+ board.
    This setting is very important.

  8. Refer to the this document, 'Sensor Configuration' which details all 10 sensor connections on the GrovePi+ board:
  9. Now that you know how to edit a Node-RED flow on a Raspberry Pi, find the IP address of the Somsen 301 Teaching Station, make sure Professor Paulson has Node-RED running on it.
  10. Using the BuzzerDigital flow, create a new flow that controls the Relay on the Somsen 301 Teaching Station.
    You should be able to turn the Teaching Station relay on and off, thereby illuminating the string of lights connected to it.

Raspberry Pi with GrovePi+ Sensors and Actuators

All 38 student workstations and the teaching station in the Somsen 301 lab have the same configuration.
Raspberry Pi Model 3 B+, GrovePi+ board with the same 10 sensors and actuators wired up to the same ports as indicated below.
All 4 wire cables are identical.
Cables must not be inserted or removed unless RPi system is completely shut down.
The example programs are written in Python, but other languages can be used.
